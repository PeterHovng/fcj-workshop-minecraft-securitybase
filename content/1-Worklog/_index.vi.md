Chương trình thực tập được thực hiện trong **12 tuần** (**20/04/2026 – 12/07/2026**) theo chương trình **AWS First Cloud Journey** của **Amazon Web Services (AWS) Việt Nam**.

Trong suốt quá trình thực tập, nội dung được chia thành **03 giai đoạn chính**:

- **Tuần 1 – 6:** Học tập và thực hành các kiến thức nền tảng về AWS Cloud, bao gồm Cloud Computing, AWS Global Infrastructure, Identity and Access Management (IAM), Amazon VPC, Amazon EC2, Systems Manager, CloudWatch, CloudTrail và GuardDuty. Đồng thời xây dựng định hướng phát triển theo **DevSecOps** và tự học bổ sung về **Cyber Security (Red Team)**.

- **Tuần 7 – 10:** Triển khai project **Minecraft Security Based on AWS**, xây dựng hạ tầng trên AWS, triển khai Minecraft Server và tích hợp các dịch vụ bảo mật như Amazon GuardDuty, Amazon EventBridge, AWS Lambda, Amazon SNS, AWS Backup, Network Load Balancer và AWS Global Accelerator.

- **Tuần 11 – 12:** Hoàn thiện project, kiểm thử toàn bộ hệ thống, tối ưu chi phí, hoàn thiện tài liệu kỹ thuật, báo cáo thực tập và **Workshop Project** theo yêu cầu của chương trình AWS First Cloud Journey trước khi kết thúc kỳ thực tập.

Ngoài các nội dung học tập và thực hành, trong quá trình thực tập còn tham gia các hoạt động chuyên môn của chương trình như **First Cloud AI Journey – Agentic AI Build Week**, **Mini Meetup – First Cloud AI Journey** (vai trò **Speaker**) và **The First Cloud Journey – AWS Việt Nam Knowledge Competition** nhằm nâng cao kiến thức chuyên môn và kỹ năng thực tế.
