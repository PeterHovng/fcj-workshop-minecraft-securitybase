This internship was carried out over **12 weeks** (**20 April 2026 – 12 July 2026**) as part of the **AWS First Cloud Journey** program organized by **Amazon Web Services (AWS) Vietnam**.

Throughout the internship, the work was divided into **three main phases**:

- **Weeks 1–6:** Studied and practiced fundamental AWS Cloud services, including Cloud Computing concepts, AWS Global Infrastructure, Identity and Access Management (IAM), Amazon VPC, Amazon EC2, AWS Systems Manager, Amazon CloudWatch, AWS CloudTrail, and Amazon GuardDuty. During this phase, a career direction in **DevSecOps** was established while expanding knowledge in **Cyber Security (Red Team)**.

- **Weeks 7–10:** Implemented the **Minecraft Security Based on AWS** project by building the AWS infrastructure, deploying a Minecraft Server, and integrating security services such as Amazon GuardDuty, Amazon EventBridge, AWS Lambda, Amazon SNS, AWS Backup, Network Load Balancer, and AWS Global Accelerator.

- **Weeks 11–12:** Completed the project through end-to-end testing, system optimization, AWS cost review, technical documentation, internship report preparation, and the completion of the **Workshop Project** required by the AWS First Cloud Journey program before the internship concluded.

In addition to technical training and hands-on labs, several professional events were attended throughout the internship, including **First Cloud AI Journey – Agentic AI Build Week**, **Mini Meetup – First Cloud AI Journey** (as a **Speaker**), and **The First Cloud Journey – AWS Vietnam Knowledge Competition**, providing valuable opportunities to strengthen both technical knowledge and practical cloud engineering skills.
